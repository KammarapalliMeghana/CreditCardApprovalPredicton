# 💳 Credit Card Approval Prediction System

A Machine Learning-based web application that predicts whether a customer's credit card application will be **Approved** or **Rejected** based on their financial and personal details.

---

## 📌 Project Overview

The **Credit Card Approval Prediction System** is an intelligent web-based application developed using **Python, Flask, Machine Learning, and Data Visualization techniques**.

The system analyzes applicant information such as income, employment status, education, family details, and loan information to predict whether a credit card application should be approved or rejected.

The project demonstrates the complete Machine Learning lifecycle, including:

- Dataset Collection and Analysis
- Data Preprocessing and Feature Engineering
- Exploratory Data Analysis (EDA)
- Model Training and Evaluation
- Flask Web Application Development
- Cloud Deployment using Render
- Software Development Life Cycle (SDLC) Documentation

# ⚙️ Installation and Setup

## Step 1: Clone Repository

```bash
git clone https://github.com/KammarapalliMeghana/CreditCardApprovalPredicton.git
```

## Step 2: Navigate to Project Directory

```bash
cd CreditCardApprovalPredicton
```

## Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

## Step 4: Train the Machine Learning Model

```bash
python model_training.py
```

## Step 5: Run Flask Application

```bash
python app.py
```

## Step 6: Open in Browser

```text
http://127.0.0.1:5000/
```
